# TKit

里面封装了很多常用的函数与类目。

## 使用方式

使用时，直接导入TKit.h即可。部分不常用的头文件并未引入到TKit.h中，需要时，需要自己引入。

```
#import "TKit.h"
```
#### 未引入TKit.h的头文件

```
TScrollView.h
```

### PodFile

```
platform :ios, '7.0'
pod "TKit"
```
